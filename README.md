# Simple BMI Calculator (ระบบคำนวณดัชนีมวลกาย)

Web Application ขนาดเล็กสำหรับคำนวณค่าดัชนีมวลกาย (Body Mass Index - BMI) พัฒนาด้วย HTML, CSS และ JavaScript ใช้งานง่ายและสามารถนำไปประยุกต์ใช้กับระบบลงบันทึกข้อมูลผู้ป่วยได้

## เกณฑ์การแปลผลที่ใช้
อ้างอิงตามเกณฑ์ของคนเอเชีย (Asian Criteria):
- **< 18.5** : น้ำหนักน้อย / ผอม
- **18.5 - 22.9** : ปกติ / สุขภาพดี
- **23.0 - 24.9** : ท้วม / โรคอ้วนระดับ 1
- **25.0 - 29.9** : อ้วน / โรคอ้วนระดับ 2
- **>= 30.0** : อ้วนมาก / โรคอ้วนระดับ 3

## วิธีการใช้งาน
1. Clone repository นี้ หรือดาวน์โหลดไฟล์ `index.html`
2. เปิดไฟล์ `index.html` ผ่าน Web Browser (เช่น Chrome, Safari, Edge)
3. กรอกน้ำหนัก (กิโลกรัม) และส่วนสูง (เซนติเมตร) แล้วกดคำนวณ

## การนำไปพัฒนาต่อ
โค้ดชุดนี้ถูกเขียนด้วย Vanilla JavaScript สามารถนำ Function `calculateBMI()` ไปต่อยอดใช้งานกับ Google Apps Script, AppSheet หรือนำไปรวมกับ Web App คลินิกต่างๆ ได้ทันที
